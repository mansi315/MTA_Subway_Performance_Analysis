# MTA Subway Performance Analysis

This project analyses MTA Subway Customer Journey Metrics to understand the operational factors affecting customer journey time performance.

## Project Overview
The analysis includes:
- Exploratory Data Analysis (EDA)
- One-Way ANOVA
- Multiple Linear Regression
- Machine Learning Models:
  - Random Forest
  - LightGBM
  - XGBoost
- Hyperparameter tuning with Optuna
- Model robustness testing with deterministic seeds

## Dataset
Source: NYC Open Data / Data.gov

## Tools Used
- Python
- Pandas
- Matplotlib
- SciPy
- Statsmodels
- Scikit-learn
- LightGBM
- XGBoost
- Optuna
